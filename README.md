# Business Analytics Internship Final Project on Big Sales Prediction Using Random Forest Regressor
This initiative seeks to forecast revenue for a commerce enterprise through a Random Forest Regression technique. The Random Forest methodology is applied owing to its capacity to manage intricate data collections and deliver precise estimations.


The dataset is imported from the Ybi foundation official GitHub dataset repository. This is the link https://github.com/YBIFoundation/Dataset/raw/main/Big%20Sales%20Data.csv"
